# Gaming Sentiment ML Pipeline 🎮🤖

A production-grade pipeline that fetches real-time gaming news, analyzes sentiment using Transformers, and serves results via a FastAPI.

## 🚀 Features
- **Data Ingestion:** Fetches headlines from NewsAPI.
- **AI Inference:** Sentiment analysis using `distilbert-base-uncased`.
- **API Access:** Real-time JSON endpoints for sentiment data and summaries.
- **Containerized:** Fully Dockerized for easy deployment.

## 🛠 Tech Stack
- Python 3.11
- Hugging Face Transformers
- FastAPI & Uvicorn
- Docker
- Pandas & Requests

## 📦 How to Run (Docker)
1. Build the image: 
   `docker build -t gaming-sentiment-app .`
2. Run the container:
   `docker run -p 8000:8000 gaming-sentiment-app`
3. Visit: `http://localhost:8000/docs`
