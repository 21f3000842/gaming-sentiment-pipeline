# Gaming Sentiment ML Pipeline

A production-grade pipeline that fetches real-time gaming news and analyzes sentiment using a Transformer model (DistilBERT).

## 🛠 Tech Stack
- **Language:** Python 3.x
- **Data:** NewsAPI
- **AI/ML:** Hugging Face Transformers (DistilBERT)
- **DevOps:** Git, .env for secrets management

## 🚀 How it Works
1. `ingest.py`: Fetches latest headlines from NewsAPI.
2. `analyze.py`: Performs sentiment inference on the headlines.
